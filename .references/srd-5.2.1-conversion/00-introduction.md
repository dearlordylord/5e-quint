# Converting to System Reference Document 5.2.1

This document highlights new and revised elements in each section of System Reference Document 5.2.1 (SRD 5.2.1):

- Playing the Game
- Character Creation
- Classes
- Character Origins
- Feats
- Equipment
- Spells
- Rules Glossary
- Gameplay Toolbox
- Magic Items
- Monsters
- Monsters A-Z
- Animals

Those elements were first introduced in the *Player's Handbook* (2024), *Dungeon Master's Guide* (2024), and *Monster Manual* (2025).

Each element has one or more of the following tags:

- **New Name.** A rule that has a new name. Unless stated otherwise, the rule hasn't changed apart from its name.
- **New Rule.** A new rule.
- **Revised Rule.** A rule that appeared in SRD 5.1 but that has new functionality.
- **Omitted Rule.** A stat block omitted in SRD 5.2.1 with recommended replacements.

## Using This Guide

When updating content from SRD 5.1 to SRD 5.2.1, follow these steps:

1. **Apply New Names.** Check the new names here and update any renamed terms in your content accordingly.
2. **Check for Rules Changes by Content Type.** Consider the kind of content you're updating, and check for changes to the kind of rules it depends on.
3. **Update Capitalization.** Many more game terms are capitalized in SRD 5.2.1 than in 5.1; update their capitalization in your content to match.
